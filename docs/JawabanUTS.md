Ketika volume data dalam basis data bertambah dan mendekati kapasitas maksimum, akan terjadi beberapa dampak terhadap kinerja sistem:

Penurunan kecepatan akses data:
Query menjadi lebih lambat karena jumlah data yang harus dipindai meningkat. Proses pencarian, pengurutan, dan join antar tabel memerlukan waktu lebih lama.

Konsumsi memori dan CPU meningkat:
Server basis data membutuhkan lebih banyak sumber daya untuk memproses permintaan.

Efisiensi pengelolaan menurun:
Proses backup, restore, dan replikasi data memakan waktu lebih lama.

Fragmentasi data meningkat:
File penyimpanan database menjadi terfragmentasi sehingga memperlambat pembacaan data.
