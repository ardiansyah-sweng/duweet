Dampak Peningkatan Volume Data pada kinerja Sistem
1. Penurunan Kecepatan Akses Data (Query Performance Degradation)
   Peningkatan volume data memperlambat kinerja query (permintaan data) karena :
   - Peningkatan Beban I/O (Input/Output)
   - Pemindaian Data lebih lama (Longer Scans)
   - Indeks yang Kurang Efisien
2. Inefisiensi Pengelolaan Data (Data Management Ineffeciency)
   Operasi pengelolaan data (DML) seperti INSERT, UPDATE dan DELETE juga terpengaruh :
   - Peningkatan Locking dan Contention
   - Waktu pemeliharaan yang panjang
   - Konsumsi Sumber Daya
3. Solusi Mitigasi (Peningkatan Efisiensi Pengelolaan)
   Untuk mengatasi dampak negatif volume data yang besar:
   - Optimasi Query
   - Indexing yang strategis
   - Skalabilitas Vertikal
   - Skalabilitas Horizontal
